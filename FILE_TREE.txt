QUICK START
===========
1. Read INDEX.md
2. Create icon.png and splash.png
3. Get Thirdweb Client ID
4. Run setup.sh

6. Follow DEPLOYMENT_GUIDE.md

===============
1. Test locally
2. Push to GitHub
3. Deploy to Vercel
4. Configure env vars
5. Turn off Deployment Protection
6. Generate account association
7. Update minikit.config.ts
8. Test with preview tool
9. Launch on Farcaster!
